# HireSense AI

## AI-Powered Resume Analyzer & Smart Recruitment Platform

### Overview
HireSense AI is a full-stack MERN application that helps candidates improve their resumes using AI-powered analysis while enabling recruiters to efficiently identify the best candidates.

---

## Tech Stack

### Frontend
- React.js
- React Router
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js
- JWT Authentication

### Database
- MongoDB Atlas

### AI
- Google Gemini API (or OpenAI API)

---

## Modules

- Candidate Portal
- Recruiter Portal
- Admin Portal
- Resume Upload
- Resume Analysis
- Job Matching
- Analytics Dashboard
